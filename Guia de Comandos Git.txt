git init -> Criar o repositorio local

git remote add origin "link do github"

git add . -> Adicionar alterações feitas na pasta

git commit -m "Mensagem de commit"

git push origin main -> Empurra as alterações para o github.

*agora estamos na branch nova

git checkout -b nome da branch

git merge nome da branch -> Trás o conteudo da branch que citou o nome para a que estamos. (importante confirmar em qual branch esta)
